# vue-movies-app
> Movies & TV Shows web app developed with [Vue.js 3](http://vuejs.org/guide/), [Vuex](https://github.com/vuejs/vuex) and [Vue-router](https://router.vuejs.org/).

## Clone
```
$ git clone https://github.com/mobeen7asif/vue-movies-app.git

cd vue-movies-app
```

## Install
```
$ npm install
```

## Develop

```
$ npm run serve
```
